# Mirror Tile Challenge

An interactive geometry application designed to test and teach concepts of mathematical symmetry through tile manipulation.

## Core Features
* **Dynamic Grid:** A $4 \times 4$ grid interface utilizing square tiles split diagonally into two contrasting colors.
* **Symmetry Verification:** Algorithmic checks for both line symmetry (across 4 distinct axes) and rotational symmetry.
* **Progressive Complexity:** Procedurally generated challenges that transition from intuitive patterns to complex, asymmetric illusions.

## Mathematical Concepts Explored
* **Reflectional Symmetry:** Verification of identical halves across horizontal, vertical, and diagonal planes.
* **Rotational Symmetry:** Order of rotational symmetry determination ($90^\circ$, $180^\circ$, $270^\circ$).
